== Description ==
 WP Modify Login Layout: A very simple plugin to modify login page of WordPress.

1.Upload the  plugin to your blog and just activate it.
2.Goto Settings and then open Change Login Page Layout and make changes as needed.

That's all !!! I hope you enjoy the plugin.
